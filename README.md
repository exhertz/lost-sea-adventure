## Background
EN:
This game is a clone of the mobile game DOKDO, which was very popular in the past. Here is her page on reddit: [link](https://www.reddit.com/r/DOKDOgame/).  
I decided to repeat the idea and present it as a graduate work.

RU:
Данная игра является клоном мобильной игры DOKDO, которая была в прошлом очень популярной. Вот ее страница на reddit: [link](https://www.reddit.com/r/DOKDOgame/).  
Я решил повторить задумку и представлю ее на дипломной работе. 

## What's interesting?
EN:
- Based on the "non-standard" Godot v4 engine.3;
- Awesome wave physics and water appearance;
- A self-modeled ship, albeit with the use of free assets;
- All the decoration is done in the style of sea adventures;
    
RU:  
- Основано на "нестандартном" движке Godot v4.3;
- Обалденная физика волн и внешний вид воды;
- Самостоятельно смоделированный корабль, хоть и с применением бесплатных ассетов;
- Всё оформление сделано в стиле морских приключений;
